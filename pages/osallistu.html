Osallistu tai luo oma

Halutessasi voit osallistua oppikirjojen jatkokehittämiseen tai luoda omat versiosi kirjoista. Kirjojen lähdetiedostot ovat julkisena GitHub-palvelussa:

  • Lukion oppikirjat
  • Yläkoulun oppikirjat
  • Muut oppikirjat

Kirjasta löytyvien virheiden raportointi

Koska kirja ei vielä ole täysin valmis, löytyy siitä todennäköisesti myös virheitä. Voit auttaa ilmoittamalla löytämäsi virheet kirjan GitHub-sivun Issues-kohdassa

Kirjan kirjoittamiseen osallistuminen

  1. Luo itsellesi ilmainen GitHub-tunnus, jos sinulla ei sellaista vielä ole.
  2. Asenna Git-versiohallintaohjelma koneellesi (Windows / Mac)
  3. Asenna LaTeX-kääntäjä koneellesi
    • Tätä tarvitaan, sillä kirja on kirjoitettu LaTeXia käyttäen, jota käytetään paljon matemaattisen tekstin ladontaan
    • Windows-käyttöön MikTeX
    • Mac-käyttöön MacTeX
    • Linux-käyttöön TeX Live
  4. Asenna tekstieditori LaTeX-käyttöä varten. Esimerkiksi Texmaker on hyvä vaihtoehto ja toimii kaikissa käyttöjärjestelmissä.
  5. Kirjaudu sisään GitHubiin ja kloonaa oppikirjaprojekti omaan GitHub-työtilaasi klikkaamalla Fork-nappia osoitteessa https://github.com/avoimet-oppimateriaalit-ry/vapaa-matikka/
  6. Tee nyt kloonaus omalle koneellesi seuraavasti:
    • Avaa Git Shell omalla koneellasi (asensit sen vaiheessa 2)
    • Suorita Git Shellissä komento: git clone https://github.com/<oma-tunnus>/vapaa-matikka/
  7. Nyt voit muokata kirjaa lokaalisti niin paljon kuin sielu sietää :)
    • Muokkaamiseen tarvitset kylläkin LaTeX-osaamista, mutta onneksi netistä löytyy ohjeita.
  8. Git-versionhallintakomennoista tarvitset ensisijaisesti näitä:
    • git pull
    • git commit -a -m "Selitä mitä muokattu..."
    • git push

Osallistu keskusteluun IRC-kanavalla

IRC eli Internet Relay Chat (eri asia kuin IRC Galleria!) on jo kauan käytössä ollut teknologia, jota edelleen hyödynnetään paljon esimerkiksi avoimen lähdekoodin ohjelmisto- ja sisällönkehitysprojekteissa reaaliaikaisena kommunikointikanavana. Kuvittele Facebook-chat, mutta niin, että osallistumiseen ei vaadita jäsenyyttä tms. Voit tulla ja mennä kuten haluat.
